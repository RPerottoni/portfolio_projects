DATA SCIENCE PROJECTS

Customer Segmentation for Loyalty Program

Created a customer segmentation model to support loyalty program strategies by identifying behavioral patterns in a retail customer base. Using purchase frequency, recency, and monetary value, the goal was to classify customers into meaningful groups for targeted marketing actions.

Applied unsupervised learning techniques (K-Means, GMM, Hierarchical Clustering) on an RFM matrix to group customers with similar profiles. Evaluated the quality of clusters using Silhouette Score and embedding visualization techniques.

This project enabled a data-driven understanding of customer value and behavior, helping guide retention strategies and campaign personalization.

Main Tools

  • Python, Pandas, Jupyter Notebook
  • RFM Matrix, K-Means, GMM, Hierarchical Clustering
  • Scikit-learn, Matplotlib, Seaborn
  • Clustering Evaluation Metrics: Silhouette Score, Distance Analysis
  • More

InStyle Net Promoter Score Prediction

Developed a machine learning model to predict customer dissatisfaction for a retail clothing company, based on behavioral and survey data. The goal was to anticipate negative customer experiences and enable the business to take proactive retention actions.

The solution involved data cleaning, exploratory analysis, feature engineering, and classification modeling (Random Forest, Logistic Regression, XGBoost). The model achieved strong performance metrics and allowed segmentation of customers into promoters, passives, and detractors.

This project demonstrates the use of Data Science in improving customer experience and reducing churn through targeted interventions.

Main Tools

  • Python, Pandas, Jupyter Notebook
  • Scikit-learn, XGBoost, Classification Algorithms
  • Exploratory Data Analysis, Feature Engineering
  • Performance Metrics: Accuracy, Precision, Recall, F1 Score
  • More

Health Insurance Cross-Sell Prediction

Designed a machine learning model to predict which existing health insurance customers are most likely to purchase a vehicle insurance policy. This helps the company focus its sales efforts and marketing budget on the highest-converting leads.

Performed detailed exploratory analysis, feature engineering, and hyperparameter tuning using KNN, XGBoost, and Logistic Regression. Deployed the model as an API integrated with Google Sheets for real-time use by the business team.

The solution improves lead conversion rates and optimizes decision-making by ranking customers based on likelihood to buy.

Main Tools

  • Python, Pandas, Jupyter Notebook
  • EDA, Feature Engineering, Model Tuning (Optuna)
  • Classification Models: XGBoost, Logistic Regression, KNN
  • Flask API, Google Sheets Integration
  • Performance Metrics: Confusion Matrix, ROC Curve, Precision, Recall
  • More

Rossmann Store Sales Forecast

Developed a time series regression model to predict sales for over 1,000 stores of a large retail chain (Rossmann) over a six-week period. The objective was to support inventory planning and resource allocation across all locations.

Applied the CRISP-DS methodology, conducted EDA, hypothesis testing, and feature selection using Boruta. Trained multiple models including XGBoost, Random Forest, and Lasso Regression. Deployed the final model with a Telegram Bot for on-demand predictions per store.

This project demonstrates a full ML pipeline from business understanding to deployment, enabling real-time interaction with model results.

Main Tools

  • Python, Pandas, Jupyter Notebook, Scikit-learn
  • XGBoost, Random Forest, Linear/Lasso Regression
  • Feature Engineering, Boruta, EDA
  • Flask, Telegram Bot, Deployment on Heroku
  • Regression Metrics: RMSE, MAE, MAPE
  • More

Automated Web Scraping for Product Tracking

Developed a data pipeline to automatically collect and store fashion retail data from a target e-commerce website. The goal was to monitor prices, availability, and product details to support competitive analysis and dynamic pricing strategies.

Implemented the pipeline using Python (BeautifulSoup, Regex) and stored data in a local SQLite database. The entire process was automated via Cronjob to run on a schedule, allowing for consistent data collection over time.

This project highlights my ability to create scalable data collection systems for continuous market monitoring.

Main Tools

  • Data Cleaning, Logging, Task Automation
